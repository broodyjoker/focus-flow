## ✨ Key Features

* 🎯 **Zone Mode (Deep Focus):** A minimalist, distraction-free dashboard. Hand-pick your most critical tasks and focus on them one by one.
* ⏱️ **Integrated Pomodoro Timer:** Built-in focus timer that syncs seamlessly with your active tasks.
* 📱 **Progressive Web App (PWA):** Fully installable on Windows, macOS, Android, and iOS. Works 100% offline with zero internet connection required.
* 🗂️ **Advanced Task Management:** * Break down large tasks into manageable **Sub-tasks**.
    * Organize by customizable **Categories**.
    * Set priority levels and energy requirements.
* 🔔 **Smart Notifications & Audio:** Gentle audio cues and native push notifications to guide you through focus and break sessions.
* 🎨 **Beautiful UI/UX:** A buttery-smooth, dark-themed interface crafted with Tailwind CSS for a premium native-app feel.

## 🚀 How to Install & Use

### Option 1: Install as a Native App (Recommended)
1. Visit the [Live Website](https://broodyjoker.github.io/focus-flow/).
2. Open the **Settings** menu inside the app.
3. Click the **"Install App ⬇️"** button.
4. The app will be installed on your device for fast, offline access!
